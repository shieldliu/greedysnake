# Greedy Snake — 贪吃蛇

这是一个用 C 语言编写的贪吃蛇游戏，运行在 Linux 终端中。本项目是学习 C 语言编程实践的一部分，旨在通过编写小游戏来巩固 C 语言基础。

## 项目说明

本项目是一个 C 语言学习练习系列，每个子目录或源文件对应一个小游戏。目前包含：

| 游戏 | 文件 | 说明 |
|------|------|------|
| 贪吃蛇 | `snak.c` | 贪吃蛇游戏，支持 WASD 方向控制和 ESC 退出 |

更多游戏正在开发中……

## 运行环境

- **操作系统：** Linux（推荐）
- **编译器：** GCC 或任何兼容 C99 的 C 编译器
- **构建工具：** CMake（推荐）或直接使用 GCC

## 快速开始

### 使用 CMake 构建

```bash
mkdir build && cd build
cmake ..
make
./snak
```

### 直接使用 GCC 编译

```bash
gcc snak.c -o snak
./snak
```

### MSDOS 兼容版本

如果需要编译 MSDOS 版本，定义 `MSDOS` 宏即可：

```bash
gcc snak.c -o snak -DMSDOS
```

## 操作方式

| 按键 | 操作 |
|------|------|
| <kbd>W</kbd> / <kbd>w</kbd> | 向上移动 |
| <kbd>A</kbd> / <kbd>a</kbd> | 向左移动 |
| <kbd>S</kbd> / <kbd>s</kbd> | 向下移动 |
| <kbd>D</kbd> / <kbd>d</kbd> | 向右移动 |
| <kbd>ESC</kbd> | 退出游戏 |

## 游戏规则

- **目标：** 控制蛇头移动，吃到食物（`O`）让蛇身增长
- **结束条件：** 蛇头碰到墙壁（超出游戏区域边界）即游戏结束
- **显示符号：**
  - `X` — 蛇身
  - `O` — 食物
  - `=` — 上下边框
  - `|` — 左右边框
